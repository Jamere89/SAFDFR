# SAFDFR - San Andreas Fire & EMS Roleplay Mod

A comprehensive GTA V mod for RAGEPluginHook that adds realistic Emergency Medical Services (EMS) and Fire Department gameplay.

## Features

### Core Systems
- **Realistic Dispatch System** - Call generation based on real emergency scenarios
- **Medical Diagnosis & Treatment** - Full EMS equipment including BP cuffs, glucose tests, defibrillators
- **Mobile Data Terminal (MDT)** - In-game communication and dispatch management
- **Radio Communications** - Realistic dispatch audio with authentic emergency tones
- **Shift & Jurisdiction Management** - Work realistic shifts across multiple jurisdictions
- **AI Ambient Response** - Watch other emergency vehicles respond to calls
- **Realistic Call Timing** - Call volumes and response times based on real data

### Medical Features
- Patient diagnosis and vitals monitoring
- Medication administration
- Wound treatment
- Patient transport and hospital delivery
- Realistic medical protocols

### Firefighting Features
- Structure fires and vehicle fires
- Extrication scenarios
- Hazmat responses
- Rescue operations

## Installation

1. Install RAGEPluginHook on your GTA V installation
2. Download the latest SAFDFR release
3. Copy DLL to your plugins folder
4. Launch GTA V through RAGEPluginHook

## Requirements

- GTA V (Latest version)
- RAGEPluginHook
- .NET Framework 4.8+

## Usage

Press `F8` to open the MDT interface and begin your shift.
